# Web Development Unit 2

## Markup Languages

### Core Concept

**Markup Languages** - A text-encoding system consisting of symbols inserted into a text document to control its structure, formatting, and relationship between the different parts.

### Types of Markup Languages

#### XML (Extensible Markup Language)

**XML** - A human and machine-readable language used for marking up and describing simple and complex data structures.

#### JSON (JavaScript Object Notation)

**JSON** - An open standard language and file format that uses human-readable text to store and transmit data using the JavaScript syntax for describing objects.

#### SVG (Scalable Vector Graphics)

**SVG** - An XML-based language for defining vector graphics designed for the web.

## Data Processing

### Serialization

**Serializing** - The process of converting data from one format to another for storage or transmission.
